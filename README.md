# group_project_2_api

В данном проекте мы решили рассмотреть новости, которые влияют на котировки, чтобы отследить как и из-за чего они меняются. 

Нашей бизнес-задачей будет предсказание смены цены в будущем для рассматриваемых компаний. 

Изначально наша команда рассмотрела сайт https://stockscan.io/stocks/AAPL/price-history, мы спарсили данные (Parsing_requests_Sasha&Vera.ipynb), но поняли, что информации с данного сайта нам не хватает. Мы пришли к выводу, что он нам не подходит и начали искать другой. 

В итоге нами был выбран популярный сайт https://finance.yahoo.com/ с котировками, в котором есть информация о зарубежных выбраннных нами компаниях. 

Так как у yahoo нет официального API, мы решили парсить информацию с сайти файл Parsing_Vera_Rena.ipynb

Далее мы начали искать агрегаторы новостей. В файлах finhub_research_api_Tim.ipynb api_research_TheGuardian_Tim.ipynb мы через Api создали датасет с новостями (сразу добавляя только подходящие нам новости, которые реально влияют на цену акций).

Далее мы сделали EDA (GP2_EDA_Darya.ipynb), в которой проанализировали все выше полученные датасеты, предварительно их обработав. 

Также мы сделали файл mainbook.ipynb, который позволяет вывести всю необходимую информацию о принятии решения о покупке акций, а именно и в том числе тикер, логотип, оценки аналитиков, открытие биржи (открыта ли в данный момент?)

На каждом этапе мы логировали наши дейтсвия (журнал логов (py_log.log)). Также через файл logger_config.py можно отключить логирование, в файле вместо logs = True поставить False, аналогично для детализации (details = False)
